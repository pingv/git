Git Command Guide

Hover over each part of the command to see what it does.

List all branches:

git Command to invoke Git version control branch Manage branches (list, create, or delete) -a Show all branches (local and remote)

Create and switch to a new branch:

git Command to invoke Git version control checkout Switch branches or restore working tree files -b Create a new branch and switch to it feature-name Name of your new branch

Stage all changes:

git Command to invoke Git version control add Add file contents to the staging area . Stage all files in the current directory
